# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.5] - 12/12/2025

### Changes

- Switched to Netlify functions for seamless backend integration
- Fixed API Key generation user layout

## [0.1.4] - 11/09/2025

### Added

- Added Image upload support for profile picture and project cover
- Uses S3 for object storage
- Added edit functions for Employment and Projects

## [0.1.3] - 11/08/2025

### Added

- Manage Access Key and Manage PromptTab
- Fix route endpoint on /projects for fetching via API key
- Removed Icon input on add skill modal for unified flow.
- Added system_prompt table on database

## [0.1.2] - 11/05/2025

### Added

- Functional Manage Employment tab
- Added animation on card pop ups, hover, to remove static feel
- Added DELETE function on skills, projects, and employments
- Mobile-responsiveness and layout changes

## [0.1.1] - 11/04/2025

### Added

- Added Middleware and Authentication
- Functional Manage Profile and Manage Projects tab
- Collapsible Sidebar
- Dark Mode/Light Mode
